# Sound Burner

## Description
This project is a static landing page for a music streaming service.
The layout is built based on a design mockup with a focus on pixel-perfect implementation.

## Live Demo
https://egorzhulanovbar0313.github.io/Sound-Burner/

## Design
This project is based on a Figma template: https://www.figma.com/design/0x0YwpafGSw99ctuoWvL8q/sound-burner-front-end?node-id=1-173&t=iU4N70kSlELiuAiq-1

## Tech Stack
HTML, CSS

## Features
- Clean and structured layout
- Semantic HTML markup
- Pixel-perfect design implementation

## Limitations
- Not responsive (desktop only)
- No JavaScript functionality

## How to Run
Open index.html with Live Server
